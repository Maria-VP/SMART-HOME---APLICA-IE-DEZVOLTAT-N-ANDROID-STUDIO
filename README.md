# Aplicație Practică

## Descriere

Acest proiect prezintă dezvoltarea unui sistem Smart Home care monitorizează în timp real parametri de mediu, precum temperatura, 
umiditatea, concentrația de gaz și mișcarea cu ajutorul senzorilor.

### Activități
- MainActivity – prima pagină de bun venit.
- LoginActivity – autentificarea utilizatorilor.
- CreateAccount – crearea unui cont nou.
- MainPage – pagina principală cu toate meniurile aplicației.

### Fragmente
- HomeFragment – ecranul principal.
- MonitorFragment – monitorizarea grafică a senzorilor.
- HistoryFragment – afișarea istoricului.
- Profile – gestionarea profilului utilizatorului.
- Settings – configurarea aplicației.

### Clase auxiliare
- ConnectionClass – gestionarea conexiunii cu baza de date/serverul.
- HistoryItem – model pentru elementele din istoric.
- HistoryRecyclerViewAdapter – adaptor pentru afișarea istoricului în RecyclerView.

### Resurse
- AndroidManifest.xml – configurarea aplicației.
- res/drawable – iconițe și resurse grafice.

## Livrabile

- Codul sursă complet al aplicației (fără fișiere binare compilate).
- Fișierele de configurare Gradle.

## Cerințe

- Android Studio (cu Android SDK instalat)
- JDK
- Conexiune la internet pentru descărcarea dependențelor Gradle la prima compilare

2. Deschideți proiectul în Android Studio.

## Pașii de instalare

1. Conectați un dispozitiv Android sau porniți un emulator.
2. Din Android Studio selectați dispozitivul.
3. Apăsați Run

## Lansarea aplicației

După instalare, aplicația poate fi lansată:
- din Android Studio folosind butonul Run;
- sau direct de pe dispozitivul Android, selectând pictograma aplicației.
